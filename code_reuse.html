This webpage will allow you to search websites by their source code. Every web developer has his own way of solving problems or a favorite syntax. Parts of the code will almost always be reused between their projects. As front-end code of a website is generally public, services like mine will use it to their advantage and make some sort of source code search engine. When you search a domain name in the fields below, our server will download its source code and compare it to our >1 million database of scraped webpages. If it finds any similarities between another webpage, it can usually be assumed it was made by the same person.
